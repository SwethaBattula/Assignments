📊 Data Handling & Exploratory Data Analysis
📌 Overview

This module focuses on data handling, preprocessing, and exploratory data analysis (EDA) using Python.

The objective is to understand how raw data is transformed into a clean, structured, and analyzable format, which is a crucial step before applying machine learning models.

This folder includes hands-on implementations using:

NumPy for numerical operations
Pandas for data manipulation and cleaning
EDA techniques for extracting insights
